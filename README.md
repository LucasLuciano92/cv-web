# CV Web - Richard Tex Tex

## Descripción

Este proyecto consiste en un CV web desarrollado con HTML, CSS y JavaScript como parte de un trabajo práctico de programación.

El sitio presenta información personal, experiencia, educación, habilidades y un formulario de contacto.

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

---

## Funcionalidades

* Navegación por secciones
* Diseño responsive
* Modo oscuro
* Validación de formulario
* Botón interactivo
